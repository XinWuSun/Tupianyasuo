# 图片压缩工具

这是一个简单易用的在线图片压缩工具,具有以下功能:

- 支持上传 PNG、JPG 等格式的图片
- 可以预览原图和压缩后的效果
- 显示压缩前后的文件大小对比
- 支持下载压缩后的图片
- 支持调整图片尺寸
- 采用简洁优雅的苹果风格设计

## 在线使用

访问：https://你的用户名.github.io/image-compressor

## 使用说明

1. 点击上传区域或"选择图片"按钮上传图片
2. 通过滑块调节压缩质量（0-100%）
3. 可选择调整图片尺寸：
   - 输入期望的宽度或高度
   - 可选择是否保持原始宽高比
4. 实时预览压缩效果
5. 点击"下载"按钮保存压缩后的图片
